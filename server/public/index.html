API Integration

Our app provides a comprehensive set of API endpoints for restaurant data, platter recommendations, and addon services.

GET /api/restaurants - Get all restaurants with filtering
GET /api/restaurants/:id - Get restaurant details
GET /api/restaurants/:id/platters - Get available platters
GET /api/restaurants/:id/platters/recommendations - Get AI-recommended platters
GET /api/restaurants/:id/addons - Get available add-on services
POST /api/auth/send-otp - Send OTP verification code
POST /api/auth/verify-otp - Verify OTP code
